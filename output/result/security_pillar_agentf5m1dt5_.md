| **Detection & Monitoring** | AWS Config                                                         | For continuously monitoring and recording your AWS resource configurations and evaluating them against desired configurations. Helps in identifying misconfigurations and compliance drift.                                |
| **Detection & Monitoring** | AWS Security Hub                                                   | Aggregates security findings from various AWS services (GuardDuty, Inspector, Macie, WAF, Config) and partner solutions into a single, centralized view.                                                                |
| **Detection & Monitoring** | Centralized Logging & Analytics (e.g., S3 for CloudTrail, CloudWatch Logs Insights, or SIEM integration) | While logs are generated, a centralized repository for long-term retention, easier querying, and analysis (potentially integrating with a SIEM like Splunk or ELK) is a best practice.                                   |
